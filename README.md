# bio-site

Личный сайт-био с 3D-эффектами, кастомизацией и анимациями.

## Функции

- **3D эффекты**: Анимированные 3D объекты с использованием Three.js и React Three Fiber
- **Кастомный фон**: Дождь, снег, частицы, звезды, градиенты, шум
- **Фоновая музыка**: Встроенный плеер с возможностью вкл/выкл
- **Кастомный курсор**: Несколько стилей курсора с анимациями
- **Панель настроек**: Изменение всех параметров в реальном времени
- **Адаптивный дизайн**: Работает на всех устройствах

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Сайт будет доступен на http://localhost:3000

## Сборка для продакшена

```bash
npm run build
npm start
```

## Деплой на Vercel

1. Загрузите код на GitHub
2. Подключите репозиторий к Vercel
3. Vercel автоматически определит Next.js и настроит деплой

### Настройка Vercel KV (опционально)

Для сохранения настроек в базу данных:

1. В Vercel Dashboard создайте KV Store
2. Подключите его к проекту
3. Раскомментируйте код в `src/app/api/config/route.ts`

## Кастомизация

### Изменение аватара

Замените файл `public/avatar.svg` или `public/avatar.png` на свою картинку.

### Изменение музыки

Добавьте файл `public/music.mp3` с вашей музыкой.

### Настройки по умолчанию

Измените `src/types/config.ts` для изменения настроек по умолчанию.

## Структура проекта

```
src/
  app/              # Next.js App Router
  components/       # React компоненты
    3d/             # 3D сцена (Three.js)
    backgrounds/    # Компоненты фонов
    cursor/         # Кастомный курсор
    icons/          # SVG иконки
    music/          # Музыкальный плеер
    profile/        # Компоненты профиля
    settings/       # Панель настроек
  store/            # Zustand store
  types/            # TypeScript типы
public/             # Статические файлы
```
